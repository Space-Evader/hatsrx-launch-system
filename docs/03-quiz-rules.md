# Dr. HatsRx Quiz Rules

## Purpose

This quiz matches a user's mood, vibe, and style to a HatsRx collection and product.

---

## Questions

### 1. How are you feeling?

- bold → bold-energy-rx / confidence-rx
- focused → focus-rx
- creative → creative-flow-rx
- chill → chill-rx
- mysterious → stealth-mode-rx
- rebellious → bold-energy-rx

---

### 2. How do you want to show up?

- confident → confidence-rx
- clean → focus-rx
- artistic → creative-flow-rx
- low-key → stealth-mode-rx
- loud → bold-energy-rx
- luxury → confidence-rx / stealth-mode-rx
- street → bold-energy-rx / creative-flow-rx

---

### 3. What fit do you prefer?

- snapback
- dad-hat
- trucker
- fitted
- bucket

---

### 4. What color energy?

- neutral
- bright
- earth
- cool
- monochrome

---

### 5. Where are you wearing it?

- everyday → chill-rx / focus-rx
- night-out → confidence-rx / stealth-mode-rx
- work → focus-rx
- travel → chill-rx / creative-flow-rx
- content → creative-flow-rx / bold-energy-rx
- event → bold-energy-rx / confidence-rx

---

## Output Logic

1. Determine primary Rx Collection
2. Filter by fit preference
3. Filter by color
4. Return top matching hats

---

## Result Format

- Diagnosis
- Prescription (collection)
- Recommended products
- Dosage
- Side effects

---

## Rule

Keep it simple. No AI required for version 1.
